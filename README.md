# FAA WMT Scheduler Downloader

An automated script using Playwright to log into the FAA WMT Scheduler, navigate Okta authentication, and download the current and next pay period schedules to a JSON file.

## Prerequisites
- Python 3.8+
- Linux/macOS/Windows
